## Usando Nodejs

### Iniciar NodeJs

###  `npm init`

### Instalando modulos

###  `npm install`

### Instalando Artillery

###  `npm install -g artillery`

### Instalando 0x

###  `npm install -g 0x`

### Para iniciar 0x

###  `npm start`

### Para iniciar test con autocannon

###  `npm run test`
